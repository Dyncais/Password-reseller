Файл может требовать модификации/порезки на некоторых ОС. Формат MD5 легче всего поставить через одну библиотеку, но сама библиотека может работать некорретно. На этот случай в main и cmakelists будут комментарии у строк, которые необходимо удалить для работы только с SHA-256.


Expired хэширует SHA-256 и MD5. Для работы необходима библиотека openssl, но в разных ОС она ставится по разному. Для уменьшения рисков bash файл делать не стал. Windows либо через инсталятов, либо ручной curl. Linux и подобные через установку пакета.

Весь код можно запустить в IDE, но для эффективной работы необходимо сделать Release. В консоли директивы написать.

mkdir Release
cd Release
cmake -DCMAKE_BUILD_TYPE=Release ..
make

Должна быть папка build. В неё уже необходимо скинуть файл hash.txt (хэши из примера). Собранный проект называется SHA, его запустить для работы.
